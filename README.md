# Bathroom Tracker

A Windows desktop app for tracking student bathroom pass usage, built for Securly Pass data.

## Download

**[Download Latest Version](https://github.com/mikemcs121/Bathroom-Tracker/releases/latest)**

No installation required — just download and run the `.exe`.

---

## Features

- **Student Totals** — See total bathroom time per student across a date range, color-coded by OK / Near Limit / Over Limit
- **Pass Detail** — Browse individual pass records, filter by student name or date
- **Raw Data** — Import CSV exports from Securly Pass; view all records
- **Block Schedule** — Configure your class blocks with names, start/end times, and teacher assignments
- **Configurable Limits** — Set your own Near Limit and Over Limit thresholds in minutes
- **Auto-Updates** — The app checks for new versions on startup and downloads them automatically

## How to Use

1. Download the `.exe` from the link above
2. Run it — no install or admin rights needed
3. Go to the **Raw Data** tab and click **Import CSV from Securly Pass**
4. Select your exported CSV file
5. Use the **Student Totals** tab to see who is approaching or over their limits

## Updating

When a new version is available, the app will notify you on startup and offer to download it automatically. Your data is preserved across updates.

## Requirements

- Windows 10 or later (64-bit)
